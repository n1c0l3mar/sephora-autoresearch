# Experiment Log

## Experiment 1: Baseline Linear Regression

**Date:** April 20, 2026  

**Goal:**  
Establish a reproducible baseline model for predicting Sephora product ratings.

**Model:**  
Linear Regression

**Features:**  
Structured product features (numeric + categorical)

**Target:**  
Product rating

**Validation Metric:**  
Mean Absolute Error (MAE)

**Result:**  
- Validation MAE: 0.3825  
- Runtime: 0.07 seconds  

**Notes:**  
- This is the initial baseline model.
- It provides a benchmark for future improvements.
- Future work will focus on improving model performance through feature engineering and more complex models.
